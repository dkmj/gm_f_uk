Datakällor

Översikt över de datakällor som används i verktyget — när de senast hämtades och hur de uppdateras.

Uppdatera data

För att hämta ny data krävs Python-skript som körs via uv. Kör kommandona nedan från projektets rotkatalog: 

uv run scripts/fetch_klimatkollen.py
uv run scripts/convert_excel.py
uv run scripts/fetch_scb.py

Efter körning uppdateras JSON-filerna i mappen data/ automatiskt. Kontrollera att sources.json uppdateras med nytt datum under last_fetched.
